What is the Olympus framework ?

The Olympus framework is a custom WordPress stack which provides an object oriented development structure to code professional WordPress applications.

— An easy-to-work-with solution —

Security at a higher level

Olympus takes security seriously and helps developers avoid many common security mistakes, such as SQL injection, cross-site scripting, cross-site request forgery and clickjacking.

Its user authentication system provides a secure way to manage user accounts and passwords.

Speeds up development

Olympus was designed to help developers take applications from concept to completion as quickly as possible.

Quickly set custom post types, custom taxonomies, metabox, custom fields, administration pages, settings and more without any plugins, only built-in classes.

Load only what you need

From bundle to an other, Olympus allows you to load only what you really need simply through composer, thanks to Olympus Autoload MU plugin.
